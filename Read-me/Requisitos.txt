-- RNF --

RNF001-O design deve ser atrativo para crianças;
RNF002-e as cores selecionadas devem trazem o contraste entre quentes e frias, trazendo assim o equilíbrio visual necessário em um site para que não haja desconfortos;
RNF003-Como forma de interação com o cliente o site oferecera um jogo de quebra-cabeça;
RNF004-Além disso a interfase do site deverá ser intuitiva e de fácil manuseio;
RNF005-Compra deve exigir o mínimo de informações possíveis;
RNF006-O website deverá usar as linguagens PHP, JavaScript, a linguagem de marcação HTML,
linguagem de design gráfico CSS e framework Bootstrap;
RNF007-O site deverá apresentar um Carrossel de produtos baseado na seleção de categoria feita na realização do cadastro;
RNF008-Todos os produtos devem possuir um texto descritivo e seus detalhes;
RNF009-O banco de dados utilizado deverá ser o MYSQL;
